## SPEC-009-03: 3D Foundation and Components

### Summary
Adds Three.js 3D visualization foundation for the CLAMS Animated Explainer, including reusable components for rendering point clouds, cluster spheres, and centroids.

### Changes
- Added `three-setup.js` with scene, camera, and renderer initialization
- Added `point-cloud.js` component with sprite-based points and dual depth cues (size scaling + opacity fade)
- Added `cluster-sphere.js` component with translucent spherical boundaries and wireframe overlay
- Added `centroid.js` component with brighter/larger sprite markers
- Updated `main.js` to integrate Three.js scene with graceful degradation
- All components support GSAP animation via animatable proxy objects
