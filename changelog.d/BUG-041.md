## BUG-041: Searcher class conflict - abstract vs concrete incompatible interfaces

### Summary
Fixed Searcher class inheritance so the concrete implementation in `search/searcher.py` properly inherits from the abstract Searcher ABC in `context/searcher_types.py`.

### Changes
- Made concrete `Searcher` class inherit from `SearcherABC`
- Updated ABC method signatures to include all optional parameters from concrete class
- Removed `Searcher` from `search/__init__.py` exports to avoid circular import
- Updated import statements across codebase to import `Searcher` from `clams.search.searcher`
- Added regression tests verifying inheritance and type safety
