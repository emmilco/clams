## SPEC-058-03: CALM Orchestration CLI

### Summary
Implemented Python CLI commands replacing bash-based CLAWS orchestration scripts.

### Changes
- Added `calm task` commands (create, list, show, update, transition, delete)
- Added `calm gate` commands (check, list)
- Added `calm worktree` commands (create, list, path, merge, remove, check-conflicts)
- Added `calm worker` commands (start, complete, fail, list, context, prompt, cleanup)
- Added `calm review` commands (record, list, check, clear)
- Added `calm counter` commands (list, get, set, increment, add, reset)
- Extended `calm status` with health, worktrees, tasks, workers, counters subcommands
- Added `calm backup` commands (create, list, restore, auto)
- Added `calm session` commands (save, list, show, pending, resume, next-commands)
- Added 185+ tests for orchestration modules and CLI commands
