## SPEC-058-02: CALM Memory/GHAP

### Summary
Ported memory, GHAP tracking, code indexing, git tools, and learning features from clams to calm.

### Changes
- Added memory tools (store, retrieve, list, delete memories)
- Added GHAP tools (start, update, resolve, get_active, list, search_experiences)
- Added code tools (index_codebase, search_code, find_similar_code) - placeholders for future implementation
- Added git tools (index_commits, search_commits, get_file_history, get_churn_hotspots, get_code_authors) - placeholders for future implementation
- Added learning tools (get_clusters, get_cluster_members, validate_value, store_value, list_values)
- Added context assembly tool (assemble_context)
- Added session tools (start_session, get_orphaned_ghap, should_check_in, increment_tool_count, reset_tool_count)
- Implemented embedding layer with NomicEmbedding and MiniLMEmbedding providers
- Implemented storage layer with Qdrant and in-memory backends
- Added comprehensive test suite (60+ tests for calm package)
