## SPEC-058-04: CALM Session and Reflection

### Summary
Implemented session journaling MCP tools and CLI commands for the CALM learning loop.

### Changes
- Added `store_journal_entry` MCP tool for saving session summaries with optional log capture
- Added `list_journal_entries` MCP tool with filtering (unreflected, project, working directory)
- Added `get_journal_entry` MCP tool with optional session log content
- Added `mark_entries_reflected` MCP tool for marking entries processed and deleting logs
- Added `calm session journal list` CLI command
- Added `calm session journal show` CLI command
- Added business logic layer in `src/calm/orchestration/journal.py`
- Added comprehensive test suite (56 tests)
