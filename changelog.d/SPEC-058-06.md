## SPEC-058-06: CALM Install Script

### Summary
Added `calm install` command for automated CALM setup on fresh installations.

### Changes
- Added `src/calm/install/` module with dependency checking, template copying, JSON config merging, Docker/Qdrant management, and step orchestration
- Added `src/calm/cli/install_cmd.py` Click command with `--dev`, `--dry-run`, `--force`, `--skip-qdrant`, `--skip-mcp`, `--skip-hooks`, `--skip-server` flags
- Added `src/calm/templates/` with 15 role files, 3 skill templates, 1 workflow, and 1 config template
- Updated `pyproject.toml` with hatchling `force-include` for template bundling
- Atomic JSON writes for `~/.claude.json` and `~/.claude/settings.json` to prevent corruption
- Idempotent installation (safe to run multiple times)
- 78 tests covering all installation functions
