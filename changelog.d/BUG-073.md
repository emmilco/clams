## BUG-073: get_churn_hotspots uses wrong attribute names on ChurnRecord

### Summary
Fixed attribute name mismatches in the churn hotspots tool that caused AttributeError at runtime.

### Changes
- Changed `h.path` to `h.file_path` and `h.commit_count` to `h.change_count` in `src/calm/tools/git.py`
- Added regression test with real ChurnRecord objects
