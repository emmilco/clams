## BUG-050: Fix SessionStart hook output not injected into context

### Summary
Fixed the `session_start.sh` hook to use Claude Code's expected JSON schema so that GHAP instructions are properly injected into Claude's context at session start.

### Changes
- Updated `clams/hooks/session_start.sh` to output `{"hookSpecificOutput": {"hookEventName": "SessionStart", "additionalContext": "..."}}` instead of the custom `{"type": ..., "content": ...}` schema
- Added regression test `tests/hooks/test_bug_050_regression.py` to prevent schema regression
