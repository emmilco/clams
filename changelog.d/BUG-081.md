## BUG-081: Consolidate dual session ID systems

### Summary
Fixed conflicting session ID systems where SessionManager (UUID format),
ObservationCollector (timestamped format), and CLAUDE_SESSION_ID env var
operated independently, causing session tracking inconsistencies.

### Changes
- Added unified get_session_id() function that checks CLAUDE_SESSION_ID env var
  first, then falls back to .session_id file
- Updated SessionManager.get_current_session_id() to delegate to get_session_id()
- Changed start_session() to use env var when available, falling back to
  ObservationCollector format (session_YYYYMMDD_HHMMSS_random6) instead of UUID
- Removed uuid dependency from session module
- Added regression tests for env var priority, file fallback, and consistency
